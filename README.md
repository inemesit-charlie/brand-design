# Claude Skills

Custom skills for Claude AI focused on brand strategy and design research.

## Skills Included

### Brand Discovery & Research
A comprehensive framework for conducting strategic brand research based on Chapter 2 of [Your Book Title].

- **What it does:** Guides designers through the Brand Battlefield Framework
- **Three phases:** Deep audience research → Competitive analysis → Strategic positioning
- **Output:** Research-informed brand strategy documents
- **Best for:** Startups, redesigns, market entry strategy

**How to use:** Download `brand-discovery-research.skill` and install in Claude.

## Installation

1. Download the `.skill` file
2. In Claude, go to Settings → Skills
3. Upload the `.skill` file
4. The skill will be available in your next session

## What's Inside

- Brand Battlefield Framework (You, Audience, Competitors)
- Three Levels of Audience Understanding (Demographics → Psychographics → Emotional Drivers)
- Research methodology (Primary + Secondary research)
- SWOT Analysis framework
- Differentiator Framework (True, Provable, Relevant)
- Positioning map visualization
- Real-world Glossier case study
